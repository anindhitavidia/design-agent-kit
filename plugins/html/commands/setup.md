# /design-kit-html:setup

The HTML profile has no installation requirements.

## Steps

1. Confirm: "The HTML stack profile generates self-contained `.html` files using:
   - **DaisyUI** — semantic component library (btn, card, modal, badge, etc.) via CDN
   - **Tailwind CSS** — utility classes via CDN
   - **Alpine.js** — lightweight interactivity via CDN
   No npm, no build step — open the output files directly in your browser. Dark mode toggle is included in every prototype for free."
2. Print:
   ```
   HTML stack profile ready:
     ✅ No installation needed
     ✅ Output: self-contained HTML files (open in any browser)
     ✅ Components: DaisyUI via CDN
     ✅ Styling: Tailwind CSS via CDN
     ✅ Interactivity: Alpine.js via CDN
     ✅ Dark mode: included in every prototype
   ```
