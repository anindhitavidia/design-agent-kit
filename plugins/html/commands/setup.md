# /design-kit-html:setup

The HTML profile has no installation requirements.

## Steps

1. Confirm: "The HTML stack profile generates self-contained `.html` files using Tailwind CSS (CDN) and Alpine.js for interactivity. No npm, no build step — open the output files directly in your browser."
2. Ask: "Where should prototype HTML files be saved?" (default: `design-kit/projects/<project-name>/`)
3. Print:
   ```
   HTML stack profile ready:
     ✅ No installation needed
     ✅ Output: self-contained HTML files (open in any browser)
     ✅ Styling: Tailwind CSS via CDN
     ✅ Interactivity: Alpine.js via CDN
   ```
