## Example persona

## IT Administrator

**Role:** Manages software and access for a company of 50–500 employees.
**Goals:** Provision and deprovision access quickly; avoid security gaps.
**Frustrations:** Manual processes, lack of audit trail, vendor-specific UIs that don't integrate.
**Technical level:** Intermediate — comfortable with dashboards, not a developer.
**Key workflows:** Onboarding new hire, offboarding leaver, auditing access, connecting an IdP.
